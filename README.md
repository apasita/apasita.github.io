## 📊 Dataset Analysis — ผู้เสียชีวิตจากอุบัติเหตุทางถนน (2564–2568)

โปรเจกต์วิเคราะห์ข้อมูลผู้เสียชีวิตจากอุบัติเหตุทางถนนในประเทศไทยช่วง 5 ปี โดยใช้ข้อมูลจากแหล่ง Open Government Data ของประเทศไทย วิเคราะห์ด้วย Python และ AI 3 โมเดล

### 📌 ภาพรวมโปรเจกต์

| รายละเอียด | ข้อมูล |
|---|---|
| **แหล่งข้อมูล** | [data.go.th](https://data.go.th) — ระบบบูรณาการข้อมูลการตายจากอุบัติเหตุทางถนน (3 ฐาน) |
| **ช่วงเวลา** | ปี 2564–2568 (5 ปี) |
| **จำนวนข้อมูล** | 80,621 ราย |
| **เครื่องมือ** | Python (Pandas, Matplotlib) · Claude AI · ChatGPT · Gemini |

### 🔑 Insights สำคัญ 5 ข้อ

| # | Insight | ตัวเลขสำคัญ |
|---|---|---|
| 1 | 🔴 **มกราคมอันตรายที่สุด** — ช่วงเทศกาลปีใหม่มีผู้เสียชีวิตพุ่งสูงสุด | 1,634 ราย |
| 2 | 🔵 **ผู้ชายเสียชีวิต 3 เท่าของผู้หญิง** — สะท้อนพฤติกรรมการขับขี่เสี่ยง | 76% ชาย |
| 3 | 🟡 **รถจักรยานยนต์คือภัยอันดับ 1** — 79% ของพาหนะที่ระบุได้ทั้งหมด | 38,254 ราย |
| 4 | 🟢 **ผู้สูงอายุ 65+ เสียชีวิตมากที่สุด** — ขาดทางเลือกในการเดินทาง | 12,280 ราย |
| 5 | 🟣 **19:00 น. คือชั่วโมงมรณะ** — แสงสว่างลดลง + ความเหนื่อยล้าสะสม | สูงสุดทุกชั่วโมง |
